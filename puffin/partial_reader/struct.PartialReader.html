Struct PartialReader

source 
pub struct PartialReader<R> {
    offset: u64,
    size: u64,
    source: R,
    position_in_portion: Option<u64>,
}
Expand description

PartialReader to perform synchronous or asynchronous reads on a portion of a resource.

Fields§

§offset: u64

offset of the portion in the resource

§size: u64

size of the portion in the resource

§source: R

Resource for the portion. The offset and size fields are used to determine the slice of source to read.

§position_in_portion: Option<u64>

The current position within the portion.

A None value indicates that no read operations have been performed yet on this portion. Before a read operation can be performed, the resource must be positioned at the correct offset in the portion. After the first read operation, this field will be set to Some(_), representing the current read position in the portion.
